# **Match Them All**
Một dự án game puzzle casual lấy cảm hứng từ gameplay của Match Factory.
  
## **Gameplay Demo**
[Watch Gameplay Video](https://www.youtube.com/shorts/LdT3C1zKeKI)

## **Tính năng hiện có**
* Cơ chế gameplay chính (xử lý logic tương tác vật phẩm)
* Hệ thống power-ups
* Giao diện người dùng cơ bản
* Game loop hoàn chỉnh

## **Cách tải ứng dụng**
* Vào phần `Releases` ở phía bên phải &#8594; Tải file `MatchThemAll.apk`
* Mở file apk và cài đặt.
